# 我的Clash配置

## 仓库说明

本仓库存放 Clash Meta 配置文件，支持手机端和电脑端使用。

## 文件说明

| 文件 | 适用平台 | 说明 |
|------|---------|------|
| mobile_config.yaml | 手机端 | 整合 ip_1.bat ~ ip_6.bat 节点 |
| config.yaml | 电脑端 | Alvin9999 节点配置 |

## 使用方法

### 手机端（Clash Meta）
1. 打开 Clash Meta 应用
2. 配置 → 订阅设置 → 添加订阅
3. 粘贴手机端订阅链接
4. 启用自动更新

### 电脑端（Clash Verge）
1. 打开 Clash Verge
2. 配置 → 订阅管理 → 添加订阅
3. 粘贴电脑端订阅链接
4. 启用自动更新
